# README

## Dependencies

In order to run the application locally the following software is required:

- Ruby 2.2.2
- Bundler 1.10
- PostgreSQL 9.4.4

## Installation

1. Install dependencies
2. Go to the project directory
3. Run `bundler install`
4. Run `rake db:setup`
5. Run `rails server`
6. Open `http://localhost:3000` in the browser
